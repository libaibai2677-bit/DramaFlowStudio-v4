# 🧱 3. System Architecture (Core Design)

```
User Input
   ↓
UI Layer
   ↓
ProductController (Single Entry Point)
   ↓
UnifiedEngine (AI Core)
   ↓
Service Layer
   ├── HistoryService
   ├── RecommendationService
   ├── ExplainabilityService
   ↓
Response Assembly
   ↓
UI Rendering
```

---

# ⚙️ 4. Key Modules

## 🟢 UnifiedEngine (AI Core)
- Query enhancement
- Image search execution
- Result ranking
- Cache optimization

## 🟣 Service Layer

### HistoryService
- Tracks user behavior
- Provides context memory

### RecommendationService
- Suggests next queries
- Uses behavioral history

### ExplainabilityService
- Generates "Why this result"
- Improves user trust
