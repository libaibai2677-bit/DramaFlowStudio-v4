# 🖥️ DramaFlow AI - Installation & Onboarding Guide

## 🚀 Quick Start

### Option 1: One-click run
```bash
./start.sh
```

### Option 2: Desktop App
1. Download latest release (.exe / .dmg)
2. Install
3. Open DramaFlow AI

---

## 🌍 System Requirements
- Docker Desktop
- Node.js 18+
- 4GB RAM minimum

---

## 🧠 What you get
- AI Agent system
- Streaming responses
- Desktop UI
- Cloud-ready backend

---

## 🔄 Auto Update
The desktop app will automatically check for updates and install new versions when available.

---

## ⚙ Troubleshooting
If system fails:
```bash
docker compose down
./start.sh
```

---

## 🌟 Done
You now have a production-grade AI Agent system running locally and as a desktop app.
