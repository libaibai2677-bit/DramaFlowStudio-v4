# DramaFlow Studio v4

A modular AI content generation desktop application built with Python + PySide6.

## Vision
DramaFlow Studio v4 is designed as a pipeline-based creative system:

Prompt → Scene Builder → Image Search → Image Generation → Export

## Core Modules (Planned)
- Image Search Engine (Pexels, Pixabay, Unsplash, Bing, Google)
- Prompt Engine
- Scene Builder
- AI Image Generator (ComfyUI / FLUX / API)
- Character Consistency System
- Exporter (Markdown / Video / WhatsApp-ready content)

## Tech Stack
- Python 3.10+
- PySide6 (Desktop UI)
- Async HTTP (image APIs)
